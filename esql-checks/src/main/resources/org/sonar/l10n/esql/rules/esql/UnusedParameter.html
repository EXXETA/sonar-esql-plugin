This rule applies wheneverparameters are declared but not used.

Unused parameters should be removed.

Noncompliant Code Example

	CREATE FUNCTION B(abc CHAR) RETURNS BOOLEAN --Noncompliant
	BEGIN
	  RETURN TRUE;
	END;

Compliant Solution

	CREATE FUNCTION A(abc BOOLEAN) RETURNS BOOLEAN --OK
	BEGIN
	  RETURN abc;
	END;
